# Simple vagrant php 7.2, MySQL setup

### Installed

* Apache
* PHP 7.2
* MySQL (via MariaDB)
* Based on **ubuntu/xenial64** setup

## Description

This a basic vagrant file which installs apache and php 7.2.

> NOTE: Access mysql in terminal with **sudo mysql**
